COBOL — Quick Reference
PERFORM VARYING — Loopcopy
PERFORM VARYING WS-IDX FROM 1 BY 1 UNTIL WS-IDX > WS-MAX-ROWS MOVE WS-TABLE(WS-IDX) TO WS-WORK PERFORM PROCESS-RECORD END-PERFORM
READ FILE with AT ENDcopy
READ INPUT-FILE INTO WS-RECORD AT END SET WS-EOF TO TRUE NOT AT END PERFORM PROCESS-RECORD END-READ
DB2 CURSOR — Fetch Loopcopy
EXEC SQL DECLARE C1 CURSOR FOR SELECT EMP-ID, EMP-NAME FROM EMPLOYEE WHERE DEPT = :WS-DEPT END-EXEC. EXEC SQL OPEN C1 END-EXEC. PERFORM UNTIL SQLCODE NOT = 0 EXEC SQL FETCH C1 INTO :WS-EMP-ID, :WS-EMP-NAME END-EXEC IF SQLCODE = 0 PERFORM PROCESS-EMP END-IF END-PERFORM. EXEC SQL CLOSE C1 END-EXEC.
STRING / UNSTRINGcopy
STRING WS-FIRST-NAME DELIMITED SPACE ' ' DELIMITED SIZE WS-LAST-NAME DELIMITED SPACE INTO WS-FULL-NAME WITH POINTER WS-PTR ON OVERFLOW PERFORM HANDLE-OVERFLOW END-STRING UNSTRING WS-FULL-NAME DELIMITED BY ',' OR SPACE INTO WS-PART1 WS-PART2 END-UNSTRING
VSAM KSDS — Random Readcopy
MOVE WS-SEARCH-KEY TO VSAM-KEY READ VSAM-FILE INTO WS-RECORD KEY IS VSAM-KEY INVALID KEY MOVE 'NOT FOUND' TO WS-STATUS NOT INVALID KEY PERFORM PROCESS-FOUND END-READ

Common Error Codescopy
S0C7 = Data Exception (bad numeric data) S0C4 = Protection Exception (null pointer / bad address) S222 = Job cancelled by operator S806 = Load module not found S837 = Out of DASD space B37 = Out of space on specific volume D37 = Dataset out of space (need more extents) E37 = No more volumes to allocate
